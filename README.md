**PsicoAxioma**, es una aplicación de seguimiento emocional post-terapia potenciado por _HUME AI_.

<a name="my-custom-anchor-point"></a>
**Descripción**
PsicoAxioma es una aplicación móvil desarrollada con React Native que integra una robusta infraestructura backend construida en Node.js.
Su principal objetivo es proporcionar una herramienta de apoyo psicológico, aprovechando la vanguardia de la inteligencia artificial a través de la plataforma HUMEAI.
